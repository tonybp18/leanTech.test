# Requirements 

* Have node.js installed
    * https://nodejs.org/en/download/

---

# Run tests
## Run in headless mode
```
npx cypress run
```

### Reports
Reports will be in 
```
/cypress/reports/index.html
```
Video recording will be in
```
/cypress/videos
```
---
## Run with UI
```
npx cypress open
```
